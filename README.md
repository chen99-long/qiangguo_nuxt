## 🏗️ 命令

你可以通过以下命令来启动开发服务器：

```
npm run dev
```

你可以通过以下命令来构建`服务端渲染（SSR）`

```
npm run build
```

你可以通过以下命令静态生成`Nuxt应用程序（SSG）`

```
npm run generate
```

## 部署

正常来说只需要使用 SSG 部署静态页面即可，运行`npm run generate`并部署到服务器即可。
